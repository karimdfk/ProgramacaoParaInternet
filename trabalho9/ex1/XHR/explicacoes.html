Explicações ex1

EX1

Ao clickar no botão, o conteúdo adicional é carregado dinamicamente com Ajax

EX2

Quando o conteúdo do campo do form tem tamanho 9, ocorre uma verificação para saber se seu conteúdo é igual a algum dos conteúdos previamente registrados com php.

Caso isso ocorra, o nome da cidade é carregado dinamicamente com ajax

EX3

Ao preencher o campo CEP, caso a resposta possa ser convertida de JSON para objeto JS e o CEP seja igual a algum dos cadastrados no php, os outros campos são preenchidos dinamicamente com ajax

EX4

Ocorre o mesmo que no EX3, porém a conversão de JSON para objeto JS é manual

EX5

Ocorre o mesmo que no EX4. Além disso, um objeto JS é convertido e enviado na requisição como JSON, ele é usado para definir os campos CEP, email e apikey

EX6

Existem 2 campos de formulário que são tratados dinamicamente com Ajax.

Caso os dados enviados via JSON sejam os mesmos dos registrados no php, ocorre o redirecionamento para "home.html".

Caso contrário, aparece uma mensagem de erro para o usuário.
